### 3.3 Playwright 安装与 msedge 配置摘要

```bash
cd tests
npm init -y
npm install -D @playwright/test
```

`playwright.config.js` 关键配置：

```js
use: {
  baseURL: 'http://localhost:5193',
  channel: 'msedge',          // 使用系统自带 Edge，不下载浏览器
},
webServer: {
  command: 'dotnet run --project src/LibrarySeatReservation.Web/LibrarySeatReservation.Web.csproj --urls http://localhost:5193',
  reuseExistingServer: true,
}
```

> **不执行 `npx playwright install`**：使用 `channel: 'msedge'` 直接驱动系统已安装的 Edge，无需下载 Playwright 自带浏览器。
