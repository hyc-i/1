## C. 必改项

### C-1 · P0-01 唯一索引修复（跨阶段遗留，最高优先级）

**位置**：`src/LibrarySeatReservation.Web/DataAccess/AppDbContext.cs:45-46`

**当前代码**：
```csharp
e.HasIndex(r => new { r.SeatId, r.ReservationDate, r.StartTime, r.EndTime })
    .HasDatabaseName("IX_Reservation_Seat_Time");
```

**应改为**：
```csharp
e.HasIndex(r => new { r.SeatId, r.ReservationDate, r.StartTime, r.EndTime })
    .IsUnique()
    .HasDatabaseName("IX_Reservation_Seat_Time");
```

**同步修复**：
1. 重新生成 EF Core 迁移（`dotnet ef migrations add FixUniqueIndex`）
2. `Docs/项目任务板与迭代记录.md` 一致性修订跟踪 P0-01 状态需更新
3. `Docs/14-管理端与权限开发记录.md` 门禁检查需补充说明 P0-01 为跨阶段遗留

### C-2 · Edit Modal 楼层字段修复（本轮新增 Bug）

**位置**：`src/LibrarySeatReservation.Web/Views/Admin/Seats.cshtml:34-39`

**当前代码**：
```html
<button type="button" class="btn btn-outline-primary btn-sm"
        data-bs-toggle="modal" data-bs-target="#editModal"
        data-id="@seat.Id"
        data-seatnumber="@seat.SeatNumber"
        data-area="@seat.Area"
        data-description="@seat.Description">编辑</button>
```

**应改为**（补充 `data-floor`）：
```html
<button type="button" class="btn btn-outline-primary btn-sm"
        data-bs-toggle="modal" data-bs-target="#editModal"
        data-id="@seat.Id"
        data-seatnumber="@seat.SeatNumber"
        data-area="@seat.Area"
        data-floor="@seat.Floor"
        data-description="@seat.Description">编辑</button>
```

**注意**：`SeatItem` 模型当前没有 `Floor` 属性。需同步在 `Models/ViewModels/SeatItem.cs` 中添加 `public string? Floor { get; set; }`，并在 `SeatService.GetAllSeatsForAdminAsync` 的 Select 中赋值 `Floor = s.Floor`。
