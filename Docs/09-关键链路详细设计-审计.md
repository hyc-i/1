### D-2：L2-2 管理端搜索功能应补充设计

**问题描述**：Section 3.3 `GetAllReservationsAsync(string? statusFilter)` 只处理了状态筛选，但管理端页面（PA-02，见 05-页面卡）还有日期筛选（全部/今天/本周/本月）和文本搜索（座位号/学号）。原型 `admin-reservations.html` 也有对应搜索框。08-审计 C-2 已指出此查询路径遗漏。

**修复方案**：将 `GetAllReservationsAsync` 的签名扩展为：

```csharp
public async Task<List<ReservationItem>> GetAllReservationsAsync(
    string? statusFilter,
    string? dateFilter,     // "today" / "week" / "month" / null
    string? searchKeyword)  // 搜索座位号或学号
```

并在 Service 伪代码中补充日期范围计算逻辑和 JOIN 搜索逻辑。

**优先级**：P1（不影响核心链路闭环，但补全后更完整）
